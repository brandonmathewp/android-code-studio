## Overview
This hotfix release introduces several important improvements and fixes.

---

## Fixes
- **Fixed:** A tooling API issue that prevented dependency retrieval for projects using a version catalog (`libs.versions.toml`), affecting the new ATC templates.
- **Fixed:** Invalid configurations generated by the template system.
- **Fixed:** Duplicate dependencies being added to the classpath, which resulted in compilation errors.
- **Fixed:** Kotlin LSP terminating when opening a new file.
- **Fixed:** The default color scheme not rendering correctly.

## Minor Improvements
- **IDE:** Minor performance enhancements.
- **LSP:** [Kotlin] General improvements.
